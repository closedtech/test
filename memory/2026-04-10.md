### Barrett Division — FIXED ✅
- Старый алгоритм имел ошибки в расчёте q_hat
- Исправленная версия использует:
  - BASE = 1000 (base-1000 цифры)
  - mu = BASE^(2n) / b
  - q_hat = floor(a_top * mu_hat / BASE^(n+1))
  - Коррекция цикла для q

**Тесты Barrett прошли:**
- 100/3 → q=33, r=1 ✅
- 123456789/123 → q=1003713, r=90 ✅
- 1000000/7 → q=142857, r=1 ✅
- 999999999/123456 → q=8100, r=6399 ✅
